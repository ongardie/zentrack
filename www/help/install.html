The Quick Install Procedure:

Obtain and Install zenTrack Files

  • Download the latest version of zenTrack from source forge.
  • Untar the files.
    • On windows: Use winzip and extract them to a folder
    • On Unix: use tar -zxvf [zentrack filename].tar.gz from the command prompt
  • Move the files to the directory where they will be used.
    • The www/ directory belongs in the public_html tree. This folder can be renamed and moved as you desire.
    • THE INCLUDES/ DIRECTORY BELONGS OUTSIDE OF THE PUBLIC_HTML TREE
      (where these pages cannot be called directly, this folder can be renamed and moved as you desire)
    • The includes/misc folder contains all of the install/build/setup instructions you will need.
  • Chmod the www/ and includes/ directories, and all of their contents to 755.
  • Set the owner of these directories and files to a secure user
  • Chmod the includes/attachments directory to 666 (or 766 if you're religious)

    Configure zenTrack

  • Open the www/header.php file and change the $rootUrl and $libDir to match your system. Change any other variables here to match your preferences.
  • Open the includes/configVars.php file and change the parameters to match your database and preferences.

Build an Seed Database

  • Create a new database instance to be used by zenTrack and create a user login with appropriate permissions.
  • In includes/misc, run the build.sql file that matches your database type as a database administrator. This will create the tables and indexes
  • In includes/misc, run the seed.sql file that matches your database type. This will generate the Admin and Guest login, as well as some default system settings

zenTrack should now Run!

Login and Change Admin Password

  • Log into zenTrack as Administrator/Administrator. Immediately change the password by choosing "options" from the menu on the left, and "change password" from the subsequent screen.
  • Afterwards, visit the Admin section and configure zenTrack to your specific needs.
  • Feel free to visit the Project Support Center if you need any further assistance!
