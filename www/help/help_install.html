Installation Requirements
  • Win32 IIS, Win32 Apache, or *NIX Apache server
  • Oracle or mySQL database
  • PHP 4.0.4 or better
Tested Environments
  • Windows 2000, IIS, Oracle 8i, PHP 4.0.4
  • Linux Slackware 7.0, Apache 1.3/2.0, mySQL 3.23

Installation Components (contained in zenTrack.zip)

  • oci8_create.sql - database creation script
  • oci8_build.sql - initial configuration settings and setup
  • header.php - configuration and setup elements in here
  • zenTrack.class - contains database password and login, as well as other static settings
  • includes - folder containing classes and support scripts
  • various .php scripts - the meat of the install
Installation Instructions
  1. Unzip the zenTrack.zip file.
  2. Move the misc directory to a secure location (this contains the database build and backups of the zenTrack system)
  3. Place the remaining zenTrack files and directories as is into the web directory.
  4. Edit the header.php file, replacing $rootUrl, , and $configFile with the appropriate values.
  5. Edit the inlcudes/configVars.php file, replacing the configuration variables as appropriate for your system.
  6. Oracle Install Only:
    • Edit the misc/oci8_instance_create, misc/oci8_db_create and misc/oci8_db_build files as desired (change table names, etc)
    • Log into the oracle database
    • Run misc/oci8_instance_create.sql to intall the database instance
    • Run misc/oci8_create.sql to install the database tables
    • Run misc/oci8_build.sql to install the configuration files and start data
    • Copy the files from misc/oracle_classes as directed in the misc/README-oracle8i_conversion
    • Edit the newly copied configVars.php appropriately
  7. MySQL Install Only:
    • Edit the mysql_db_create and mysql_db_build files as desired (change table names, etc)
    • Log into the mysql database
    • Create a database instance and assign appropriate permissions
    • Run zenTrack/misc/mysql_install_files/mysql_db_create to create tables needed
    • Run zenTrack/misc/mysql_install_files/mysql_db_build to populate configuration variables
  8. Open a web browser and type the url for the zenTrack directory (access index.php)
  9. Log in as Administrator/Administrator.
  10. Immediately change the password for the admin user!! This is accomplished by choosing the "Options" bar and selecting "Change Password".
  11. Enter the Admin Menu and select zenTrack Config.
  12. Set zenTrack options as needed.
  13. Return to the Admin Menu and add users and access as desired.
  14. zenTrack is now ready for use. Delete the test data by the sql interface, or by accepting and closing the tickets when ready.
